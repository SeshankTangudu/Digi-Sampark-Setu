## Smart India Hackathon 2026 Project

Digi-Sampark Setu is a full-stack web-based platform developed during Smart India Hackathon (SIH) 2026 to support the implementation and monitoring of the PM-AJAY (Pradhan Mantri Anusuchit Jaati Abhuyday Yojana) scheme.

The platform improves transparency, accountability, and coordination between the Central Government, State Governments, Implementing Agencies, and Citizens through centralized monitoring, reporting, and dashboard systems.

This project was collaboratively developed as a team project during SIH 2026.

# Features

- Role-based authentication system
- Central Government dashboard
- State Government dashboard
- Agency management system
- Citizen interaction module
- Complaint and feedback management
- Scheme allocation tracking
- Monitoring and reporting system
- Responsive UI using React and Tailwind CSS

---

# Tech Stack

## Frontend
- React.js
- Tailwind CSS
- Vite

## Backend
- Spring Boot
- Java

## Database
- MySQL

# Project Objectives

- Improve transparency in PM-AJAY scheme implementation
- Enable effective monitoring and reporting
- Ensure coordination among government bodies
- Provide centralized dashboards and tracking systems
- Improve accountability across implementation levels
